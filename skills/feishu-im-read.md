# feishu-im-read

> Feishu (Lark) IM Message Reading

Read and search Feishu IM messages - group chats, direct messages, threads.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Message operations:
- Get chat history
- Read thread messages
- Search messages cross-chat
- Download images/files
- Filter by sender, time

## Use Cases

- Chat history analysis
- Message search
- Content archival
- Communication monitoring
