# feishu-create-doc

> Feishu (Lark) Cloud Document Creation

Create Feishu Cloud Documents from Markdown content.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Document creation:
- Create documents from Markdown
- Specify creation location (folder/wiki)
- Support async task queries
- Create in Knowledge Space

## Use Cases

- Automated documentation
- Report generation
- Knowledge base updates
- Template-based doc creation
