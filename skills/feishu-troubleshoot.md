# feishu-troubleshoot

> Feishu (Lark) Plugin Troubleshooting

Troubleshooting tool for OpenClaw Lark plugin issues.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Troubleshooting features:
- Common issues FAQ
- Diagnostic commands
- Account configuration check
- API connectivity verification
- Permission validation

## Use Cases

- Plugin debugging
- Authorization issues
- API failure diagnosis
- Configuration problems
