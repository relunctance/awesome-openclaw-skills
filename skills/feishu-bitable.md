# feishu-bitable

> Feishu (Lark) Bitable Multi-dimensional Table Management

Manage Feishu Bitable (多维表格) - create, query, update, and delete records.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Manage Bitable data:
- Create/update/delete apps
- Manage tables and fields
- CRUD operations on records
- Search and filter

## Use Cases

- Project management dashboards
- Team data collaboration
- Workflow automation
- Data collection and reporting

## Fields Support

27 field types supported including:
- Text, Number, Single Select, Multi Select
- Date, Checkbox, Person, Phone
- Hyperlink, Attachment, Created Time, Modified Time
