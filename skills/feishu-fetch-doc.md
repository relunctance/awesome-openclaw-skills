# feishu-fetch-doc

> Feishu (Lark) Cloud Document Fetching

Fetch and parse Feishu Cloud Documents content.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Document fetching:
- Get document title and content
- Parse Markdown format
- Handle large documents with pagination
- Process images and attachments

## Use Cases

- Content analysis
- Document backup
- Knowledge extraction
- Content migration
