# feishu-update-doc

> Feishu (Lark) Cloud Document Updating

Update existing Feishu Cloud Documents with various modes.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Document update modes:
- overwrite - Replace entire content
- append - Add content to end
- replace_range - Replace specific range
- replace_all - Global find/replace
- insert_before/after - Position insert
- delete_range - Remove content

## Use Cases

- Document automation
- Content updates
- Batch modifications
- Structured editing
