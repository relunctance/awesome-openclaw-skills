# Skill Finder

> Search and discover OpenClaw Skills

在 ClawHub 上搜索和发现 OpenClaw Skills。

**作者:** @ivangdavila  
**ClawHub:** https://clawhub.ai/ivangdavila/skill-finder

## 功能特点

- ClawHub Skill 搜索
- 按功能分类浏览
- Skill 发现和推荐

## 安装

```bash
clawhub install ivangdavila/skill-finder
```

## 使用场景

- 查找特定功能的 Skill
- 探索社区新工具
- Skill 选型对比
