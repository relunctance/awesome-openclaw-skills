# feishu-calendar

> Feishu (Lark) Calendar and Schedule Management

Manage Feishu Calendar and events - create, query, update, delete schedules.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Calendar operations:
- Create/update/delete calendars
- Create/update/delete events
- Manage attendees
- Query free/busy status
- Reply to invitations

## Use Cases

- Meeting scheduling
- Team calendar management
- Event reminders
- Busy/free time queries
