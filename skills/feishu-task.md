# feishu-task

> Feishu (Lark) Task Management

Manage Feishu Tasks - create, query, update tasks and task lists.

**Author:** OpenClaw Team  
**Repository:** https://github.com/openclaw/openclaw  
**License:** MIT

## Installation

Part of OpenClaw Lark extension. Included by default.

## Usage

Task operations:
- Create/update/delete tasks
- Manage task lists
- Add comments to tasks
- Manage subtasks
- Assign members

## Use Cases

- Task assignment
- Project tracking
- Team collaboration
- Deadline management
