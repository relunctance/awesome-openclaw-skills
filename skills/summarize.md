# Summarize

> Long text summarization tool

长文本摘要提取工具，将文章、文档或对话压缩为简洁摘要，保留关键信息。

**作者:** @neurocoder  
**ClawHub:** https://clawhub.ai/neurocoder/summarize

## 功能特点

- 支持多种文本格式摘要生成
- 可调节摘要长度（短/中/长）
- 保留关键数据和事实
- 支持多语言文本处理

## 安装

```bash
clawhub install neurocoder/summarize
```

## 使用示例

```
将以下文章压缩为3句话摘要：
[粘贴长文本]
```

## 使用场景

- 论文摘要
- 会议记录整理
- 长文档速览
- 邮件摘要
