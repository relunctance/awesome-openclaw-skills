# Summarize

> Text Summarization Tool

长文本摘要提取工具。将长篇文章、文档或对话内容压缩为简洁的摘要，保留关键信息。

## 功能特点

- 支持多种文本格式的摘要生成
- 可调节摘要长度（短/中/长）
- 保留关键数据和事实
- 支持多语言文本处理

## 使用示例

```
将以下文章压缩为3句话摘要：
[粘贴长文本]
```

## 安装

```bash
clawhub install summarize
```
